# Food Recipes App

This is a simple Flutter app that allows users to view a list of food items, recipes, and add new food items. The app has different screens for viewing all food items, favorite foods, recipes, and adding new food.

## Getting Started

Follow these steps to run the project on your local machine:

### Prerequisites

- [Flutter](https://flutter.dev/docs/get-started/install)
- [Dart](https://dart.dev/get-dart)

Project Structure
lib/ - Contains the Dart code for the app screens and components.
Home.dart - Main screen with buttons to navigate to different sections.
AllFood.dart - Screen to display all food items.
Recipes.dart - Screen to display recipe links.
FavoriteFood.dart - Screen to display favorite food items.
AddNewFood.dart - Screen to add new food items.
main.dart - Main entry point of the app.
README.md - Project documentation.
Navigation
Home Screen: Displays buttons to navigate to different sections.
All Food Screen: Lists all food items with images and names.
Recipes Screen: Displays recipe links for each food item.
Favorite Food Screen: Shows a list of favorite food items.
Add New Food Screen: Allows users to add new food items with images and recipes.
